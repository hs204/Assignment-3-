Name Web Attacks

Malware

source www.akamai.com/glossary/what-is-malware
Malware, short for malicious software, is designed to harm computers, networks, or servers. It includes ransomware, trojans, spyware, viruses, and more.
Learn more

Denial-of-Service (DoS) Attackd

source https://developer.okta.com/books/api-security/dos/what/
 DoS attacks flood a network with false requests, disrupting operations. DDoS attacks involve multiple systems, making them harder to block.
Learn more

Phishing

source https://www.valimail.com/guide-to-phishing/
 Phishing uses emails, SMS, social media, and more to trick victims into sharing sensitive information or downloading malicious files.
Learn more

Spoofing

source https://www.cisco.com/c/en/us/products/security/email-security/what-is-spoofing.html#~other-types-of-spoofing
 Spoofing disguises cybercriminals as trusted sources to steal information or install malware.
Learn more

Identity-Based Attacks

source https://www.crowdstrike.com/cybersecurity-101/cyberattacks/most-common-types-of-cyberattacks/#1.%20Malware
 Identity-based attacks target compromised identities, with attackers posing as users to bypass security measures.
Learn more

Code Injection Attacks

source https://www.invicti.com/blog/web-security/code-injection/
 Code injection attacks inject malicious code into systems to change their actions. Includes SQL Injection, XSS, and more.
Learn more

Supply Chain Attacks

source https://www.enisa.europa.eu/news/enisa-news/understanding-the-increase-in-supply-chain-security-attacks
 Supply chain attacks target trusted third-party vendors to inject malicious code into applications or compromise hardware components.
Learn more

Insider Threats

source https://linfordco.com/blog/insider-threats/
 Insider threats involve internal actors, either malicious or negligent, who pose dangers due to their access to networks and data.
Learn more

DNS Tunneling

source https://www.paloaltonetworks.com/cyberpedia/what-is-dns-tunneling
 DNS Tunneling leverages DNS queries and responses to bypass security measures and transmit data and code within networks.
Learn more

IoT-Based Attacks

source https://www.wallarm.com/what/iot-attack
 IoT attacks target Internet of Things devices, allowing hackers to control devices, steal data, and form botnets.
Learn more
